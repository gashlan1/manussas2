## Technology Stack

- **Frontend:** React 19 + Tailwind CSS 4 + shadcn/ui
- **Backend:** Express 4 + tRPC 11
- **Database:** MySQL/TiDB with Drizzle ORM
- **AI:** LLM Integration (Planner, Tracker, Briefing, Assistant)
- **Auth:** Manus OAuth
- **Storage:** S3
- **Deployment:** Docker + GitHub Actions CI/CD
